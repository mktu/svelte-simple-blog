# 動機

今までよくReactを使ってWEBアプリなどを作っていたのですが、たまには別のフレームワークを使ってみたいなと思い、前から気になっていたSvelteを使ってシンプルなブログを作成することになりました。

# このブログで用いている技術

このブログでは主に、

- [SvelteKit](https://kit.svelte.dev/)
- [Github Pages](https://docs.github.com/ja/pages/getting-started-with-github-pages/about-github-pages)
- [Newt](https://www.newt.so/)

を用いて実現しています。大まかな構成は以下のイメージとなっています。
